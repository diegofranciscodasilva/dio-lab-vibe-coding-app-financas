# FinAI - Smart Finance 🚀

Este repositório contém a documentação e o conceito do **FinAI**, um aplicativo inteligente de gestão de finanças pessoais criado como parte do desafio do **DIO Lab - Vibe Coding App Finanças**.

---

## 📋 Prompt Final (PRD - Documento de Requisitos do Produto)

Para a geração e estruturação do projeto na ferramenta **Lovable**, foi utilizado o seguinte prompt estruturado:

Create a clean, modern, and fully responsive Web App named "FinAI - Smart Finance". The app should be built with a modern UI/UX design system (like Tailwind CSS), focusing on an intuitive and accessible interface for personal financial management.

Core Features:
1. Dashboard:
   - Header with user profile and dark/light mode toggle.
   - Summary Cards: Total Balance, Monthly Income, Monthly Expenses, and Savings Goal progress.
   - Quick Action button to "Add Transaction".
2. Smart AI Assistant Panel:
   - A dedicated area where users can ask the AI for financial tips or analyze their spending habits (e.g., "How can I save more this month?").
3. Recent Transactions List:
   - A data table or card grid showing recent entries and exits (with positive/negative values and color-coding).
   - Category tags (Food, Utilities, Salary, Entertainment) with distinct icons.
4. Add Transaction Modal/Form:
   - Inputs: Description, Amount, Type (Income/Expense), Category, and Date.

Design & Tech Stack Preferences:
- Design: Minimalist, clean typography, soft shadows, and subtle gradients. High contrast for readability.
- Layout: Fully responsive (Mobile-first to Desktop).
- Interactivity: Smooth transitions between views, working forms, and interactive UI states.

## 💡 Resumo do Conceito

O FinAI - Smart Finance é uma aplicação moderna voltada para a gestão de finanças pessoais. Seu grande diferencial é a integração de um assistente inteligente capaz de analisar hábitos de consumo e sugerir metas de economia, tudo isso em uma interface limpa, intuitiva e responsiva.

Funcionalidades Principais:
Dashboard Completo: Visão rápida de receitas, despesas e metas de economia.

Assistente de IA Integrado: Dicas personalizadas de finanças e análise de gastos.

Controle de Transações: Registro ágil de entradas e saídas com categorização por ícones.
